# Skill: 美股開盤前監控 (Market Pre-Open Monitor)

## 任務目標
在美股開盤前 30 分鐘，自動檢查持股新聞與大盤走勢，並向用戶回報潛在風險。

## 監控對象
- **持股清單**: AMD, TSLA, ONDS
- **大盤指標**: S&P 500 (^GSPC), Nasdaq (^IXIC)

## 執行邏輯
1. **大盤走勢**: 
   - 檢查盤前交易或昨日收盤漲跌。
   - 若跌幅超過 0.5%，標記為「🔴 可能走低」。
2. **個股新聞**:
   - 搜尋 AMD, TSLA, ONDS 的最新 24 小時內新聞。
   - 摘要重要利多或利空消息。
3. **回報格式**:
   - [大盤概況] 漲跌幅與風險提示。
   - [個股動態] 條列各股票的重要新聞。
   - **禁止使用 HTML 標籤**（如 `<span>`, `<div>` 等），Discord 不支援 HTML。
   - 漲跌顏色以 emoji 表示：🟢 上漲、🔴 下跌、⚪ 持平。
   - 表格使用 Discord Markdown 格式（code block），例如：
     ```
     代號  | 盤前價格  | 漲跌幅
     AMD   | $194.40   | 🔴 -2.90%
     ```

## 排程設定
- **執行時間**: 每日 21:00 (台灣時間)
- **觸發 ID**: us-market-pre-check
