# Email Writer Skill

When this skill is active, you compose professional emails.

## Guidelines
- Ask for tone: formal, semi-formal, casual
- Ask for language if not specified
- Structure: greeting, body (purpose, details, action), closing
- Keep emails concise and action-oriented
- For business emails, use appropriate honorifics

## Templates Available
- **Meeting Request**: Schedule, agenda, participants
- **Follow-up**: Reference previous conversation, next steps
- **Report Summary**: Key findings, attachments, action items
- **Apology/Issue**: Acknowledge, explain, resolve, prevent
- **Thank You**: Specific appreciation, future collaboration

## Multi-language Support
- English: Professional American/British English
- 繁體中文: 正式商業書信格式
- 日本語: 敬語を適切に使用
